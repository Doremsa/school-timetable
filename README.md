# School Timetable
Interactive web application for viewing school schedules with tri-lingual support and smooth hover effects.

### Project Description
An interactive school schedule web application. When hovering over lessons, a smooth hover effect triggers, visually emphasizing the selected subject.

### Features:
* **Multi-language support:** Full localization for Russian (RU), English (EN), and Slovak (SK).
* **Interactive UI:** Table cell highlighting (hover effect) for better readability.
* **Responsive Design:** Looks great on any device.

# Školský rozvrh
Interaktívna webová aplikácia na prezeranie školských rozvrhov s podporou troch jazykov a plynulými hover efektmi.

### Popis projektu
Interaktívna webová aplikácia pre školský rozvrh. Pri prejdení myšou (hover) cez lekcie sa aktivuje plynulý efekt, ktorý vizuálne zvýrazní vybraný predmet.

### Vlastnosti:
* **Podpora viacerých jazykov:** Úplná lokalizácia do ruštiny (RU), angličtiny (EN) a slovenčiny (SK).
* **Interaktívne používateľské rozhranie:** Zvýraznenie buniek tabuľky (hover efekt) pre lepšiu čitateľnosť.
* **Responzívny dizajn:** Vyzerá skvele na akomkoľvek zariadení.
